## One-paragraph summary

Echo is a narrative game in which players live one fictional day in a stranger's life. The choices they make — which message to open first, how to respond to a friend's sting, what to do with an unexpected free hour — are silently mapped to validated psychological dimensions (Big Five, Schwartz values, attachment style). At the end, the game generates a beautiful, shareable visual portrait and short poetic description of who the player actually is. The product ships on iOS, Android, Windows, and macOS from a single codebase, with a B2B API layer that schools, therapists, and coaches can license per seat.

## Status

- **Stage:** Pre-build, founding documentation phase
- **Team:** 1 cofounder (backend / architecture) + 4 close advisors covering mobile, DevOps, ML, and backend
- **Capital ceiling (self-funded):** Up to €100,000 of founder capital
